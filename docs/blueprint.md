# **App Name**: Nohel Segovia Portfolio

## Core Features:

- Interactive Hero Section: Displays a hero section with a professional headline, impactful statistics, and clear calls to action.
- Projects Showcase: Presents featured projects with interactive modals detailing their problem, solution, stack, metrics, and links.
- Skills Visualization: Visualizes technical skills across multiple categories using animated progress bars.
- Contact Form with Integration: Implements a functional contact form with validation and EmailJS integration for direct messaging.
- Dynamic Metrics Dashboard: Provides a visual dashboard that displays project metrics using simple charts.js graphs.
- AI-Powered Code Suggestions: Provides AI powered code suggestions within the NanoEditor v3.2. The LLM will use reasoning to act as a tool when offering different code implementations to the user.
- Downloadable CV: Offers a feature to download the CV in PDF format with a prominent button.

## Style Guidelines:

- Primary color: Professional blue (#2563eb) to convey trust and expertise.
- Background color: Dark gray (#1f2937) to provide a modern and sophisticated feel.
- Accent color: Techy green (#10b981) to highlight skills in machine learning and HealthTech, setting Nohel apart from other developers. The contrast provides emphasis on CTAs.
- Body and headline font: 'Inter' for a modern and clean aesthetic.
- Utilize React Icons to represent technologies and skills in a clear and concise manner.
- Employ Framer Motion for smooth scroll effects and engaging entry animations throughout the site.
- Follow a modular structure using shadcn/ui components to ensure consistency and responsiveness across all sections.
